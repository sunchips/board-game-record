# Secret Hitler

- **Slug:** `secret-hitler`
- **Year published:** 2016
- **Designers:** Max Temkin, Mike Boxleiter, Tommy Maranges

Weimar-era hidden-role deduction. Liberals and fascists pass legislation from a shuffled deck; fascists secretly coordinate to install their candidate as Chancellor. Liberals win by enacting five liberal policies or assassinating Hitler; fascists win by enacting six fascist policies or electing Hitler Chancellor after three fascist policies are in play.
