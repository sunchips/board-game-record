# Catan

- **Slug:** `catan`
- **Year published:** 1995
- **Designer:** Klaus Teuber

Players settle a modular hex island, trading resources to build roads, settlements, and cities. First to 10 VP on their own turn wins.

## Files

- [`catan.schema.json`](catan.schema.json) / [`catan.md`](catan.md) — base game.
- [`example.json`](example.json) — example session record.

## Variants

None yet. Expansions (Seafarers, Cities & Knights, Traders & Barbarians, 5–6 Player) get their own delta schemas in this folder the first time one is played.
