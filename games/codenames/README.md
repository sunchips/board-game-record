# Codenames

- **Slug:** `codenames`
- **Year published:** 2015
- **Designer:** Vlaada Chvátil

Two rival spymasters know the secret identities of 25 agents whose codenames are on the table. Each team's field operatives try to contact all of their agents first, using only a single-word clue and a number from their spymaster per round. Reveal the assassin and your team loses instantly.

## Files

- [`codenames.schema.json`](codenames.schema.json) / [`codenames.md`](codenames.md) — base game.
- [`example.json`](example.json) — example session record.

## Variants

None yet. Re-themed standalones (_Pictures_, _Duet_, _Deep Undercover_, _Marvel_, _Disney_) are **separate games** with their own top-level folders — not variants of this one. Official word-pack expansions (e.g. _Codenames: Deep Undercover 2.0 Word Pack_) get their own delta schemas if adopted.
