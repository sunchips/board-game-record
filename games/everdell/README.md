# Everdell

- **Slug:** `everdell`
- **Year published:** 2018
- **Designer:** James A. Wilson

Players build a forest city of constructions and critters over four seasons, placing workers to gather twigs, resin, pebbles, and berries. Scoring blends base card VP, event and journey rewards, and special prosperity bonuses — highest total wins.
