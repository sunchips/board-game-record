# Scythe

- **Slug:** `scythe`
- **Year published:** 2016
- **Designer:** Jamey Stegmaier

Alt-history 1920s Eastern Europe. Players lead one of five factions deploying workers, mechs, and structures across a hex map, racing to achieve six end-game stars. Final scoring is non-linear — coins are awarded by a popularity-bracket × component grid, so the authoritative score is recorded directly.

- [`scythe.schema.json`](scythe.schema.json) / [`scythe.md`](scythe.md) — base game.
