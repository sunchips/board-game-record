# Petiquette

- **Slug:** `petiquette`
- **Year published:** 2025
- **Designer:** Thomas Sellner

A pattern-matching party game from Oink Games. Cards mix three animals, three hats, and numbers 1–5. Each round players eye a five-card line-up, hunt for the underlying logic, and secretly pick the card that best "fits" — you score when someone else picks the same card, rewarding shared intuition over any single right answer.
