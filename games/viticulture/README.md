# Viticulture

- **Slug:** `viticulture`
- **Year published:** 2013 (Essential Edition, 2015)
- **Designers:** Jamey Stegmaier, Alan Stone

Players inherit a rustic Tuscan vineyard and spend years planting vines, harvesting grapes, aging wines, filling orders, and hosting visitors. The year a player crosses 20 VP ends the game.

## Files

- [`viticulture.schema.json`](viticulture.schema.json) / [`viticulture.md`](viticulture.md) — base game (Essential Edition).
- [`example.json`](example.json) — example session record.
