# Parks

- **Slug:** `parks`
- **Year published:** 2019
- **Designer:** Henry Audubon

Hikers traverse a seasonal trail, collecting memories of the US National Parks. Over four rounds (seasons), players move along the trail gathering resources, trading them to visit parks, snapping photos, and claiming year-end bonuses. Highest VP wins.
