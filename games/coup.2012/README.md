# Coup

- **Slug:** `coup`
- **Year published:** 2012
- **Designer:** Rikki Tahta

A bluffing elimination game in the _Resistance_ universe. Each player starts with two hidden influence cards and takes actions — some true, some bluffed — backed by one of five roles (Duke, Assassin, Ambassador, Captain, Contessa). Lose both influences and you're out. Last player standing wins.

None yet. _Reformation_ (2014 expansion, adds factions and two new roles) gets its own delta schema the first time it's played.

## Disambiguation

This folder is `coup.2012/` because the title _Coup_ is also used by a 1988 Milton Bradley game. Records use `"game": "coup"` with `"year_published": 2012` — see [`docs/NAMING.md`](../../docs/NAMING.md#identical-printed-titles-for-different-games).
