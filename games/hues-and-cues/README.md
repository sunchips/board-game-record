# Hues and Cues

- **Slug:** `hues-and-cues`
- **Year published:** 2020
- **Designer:** Scott Brady

A colour-word-association party game on a 480-swatch grid. Each round a cue giver picks a secret colour, gives a one-word cue, then a two-word cue; every other player places a cone on the grid. Closer guesses earn more VP, and the cue giver earns VP for each guess in the bullseye rings.
