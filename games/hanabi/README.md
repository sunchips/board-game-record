# Hanabi

- **Slug:** `hanabi`
- **Year published:** 2010
- **Designer:** Antoine Bauza

A fully cooperative card game in which players hold their cards facing outward. The team works to build five colored fireworks (red, yellow, green, blue, white) by playing cards 1→5 in each suit, communicating only via spent hint tokens. Three misplays burn through the fuse and the show is over.

- [`hanabi.schema.json`](hanabi.schema.json) / [`hanabi.md`](hanabi.md) — base game.
