# The King Is Dead

- **Slug:** `the-king-is-dead`
- **Year published:** 2015 (2nd edition, 2020)
- **Designer:** Peer Sylvester

Post-Arthurian Britain is on the brink. Three factions — the Welsh, the Scots, and the English — vie for eight regions while players quietly maneuver followers behind the scenes. The game ends when all faction cubes are spent. The faction that holds the most regions produces the next king; the player with the most followers of that faction wins. Ties walk down to the next faction, and if no faction dominates, the Normans invade and everyone loses.
